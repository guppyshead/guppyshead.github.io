零拷贝技术

零拷贝
杂谈
发布日期:   2021-07-17
更新日期:   2021-07-17

概述

零拷贝技术，如其名字说的一样，当用户线程操作磁盘文件时，不需要将磁盘文件的数据进行多次拷贝，这样能极大提升系统性能。通常来说，零拷贝技术分为mmap和sendfile两种方式。

传统I/O

当用户线程通过传统I/O对磁盘文件进行操作时，操作步骤如下:

  1. 先从用户态切换到内核态，将磁盘文件数据从磁盘复制到内核空间的缓冲区，
  2. 再从内核态切换到用户态，将磁盘文件数据从内核空间缓冲区复制到用户进程的缓冲区

mmap技术

mmap指内存映射技术，通常用来进行文件I/O，用户进程操作文件时步骤如下:

  1. 从用户态切换到内存态，将磁盘文件数据从磁盘中复制到内核空间的缓冲区
  2. 在用户缓存中建立虚拟内存地址，其映射的是内核空间中缓冲区的数据
    使用内存映射技术的好处是，不需要在用户空间内再拷贝一份文件数据，减少文件数据对用户缓存的占用，并且由于使用的是虚拟内存地址，不同用户进程可以映射到同一块内存缓冲区。

sendfile

sendfile是Linux系统在2.1版本引入的函数，并在2.4版本进行了进一步的优化，主要为了解决网络I/O时数据的复制问题，其操作文件时步骤如下:

  1. 从用户态切换到内核态，将磁盘文件从磁盘中复制到内核空间缓冲区
  2. 在Socket缓冲区中建立内核空间缓冲区的虚拟地址映射
  3. 从内核态切换到用户态，将Socket虚拟地址映射的文件数据复制到网卡中并发送出去

Java中的应用

在Java中，NIO和RandomAccessFile均使用了mmap来对文件进行操作。下面是个使用RandomAccessFile的例子。

假设一个文件中有10亿个数字，这些数字都是0-99999之间的数字，并且每行都是一个数字，但是这些数字并没有顺序，写一个程序将这个文件中的数字进行升序排序。

因为Java中short类型的数据最大值为32768，小于99999，因此这些数字在Java程序需要使用int类型的数据进行存储。那么10亿个数字需要的存储空间为4*1000000000/1024/1024/1024=3.7GB，而程序运行过程中通常使用Integer来存储数字，实际需要的内存空间至少还要翻一倍。大部分机器的运行内存并没有这么大，因此使用传统I/O最终会出现内存溢出，那么这时可以使用RandomAccessFile来进行文件处理。

主要的逻辑分为以下几步：

  1. 将源文件分为100个文件(根据情况可以变更)，每次从源文件中取出一千万个数字进行排序，然后写入到新文件中
  2. 构造一个数组，数组中的每个元素存储的是这100个文件和当前文件读取的数字(刚开始默认读取一行数组)，根据当前文件读取的数字进行升序排序，其中null排在最后（因为null代表文件读到末尾），取排序后的第一个元素，将读取到的数字写到结果文件中，然后读取这个元素中的文件的下一行并复制给当前元素读取的数字
  3. 循环进行第二步，直到数组第一个元素读取到的数组为null，此时得到的结果文件就是排好序的文件。
    代码如下:
    import java.io.IOException;
import java.io.RandomAccessFile;
import java.nio.ByteBuffer;
import java.nio.channels.FileChannel;
import java.util.*;
public class HugeFileSort {
    //总的数据个数(10亿)
    private static Integer TOTAL_NUMBER;
    //分割的文件个数
    private static Integer SPLIT_NUMBER;
    //数组中的每个元素
    static class FileHolder {
        private RandomAccessFile file;
        private String data;

        FileHolder(RandomAccessFile file) throws IOException {
            this.file = file;
            //file操作数据的位置可能不在文件开头，这里置为文件开头
            file.seek(0);
            this.data = file.readLine();
        }
    }
    //生成源文件
    public static void wirteFile(String file) throws IOException {
        RandomAccessFile srcFile = new RandomAccessFile(file, "rw");
        TOTAL_NUMBER = 1000000000;
        SPLIT_NUMBER = 100;
        Random random = new Random();
        for (int i = 0; i < TOTAL_NUMBER; i++){
            int randomNum = random.nextInt(100000);
            srcFile.writeBytes(String.valueOf(randomNum) + System.lineSeparator());
        }
        srcFile.close();
    }
    //切割源文件
    public static List<RandomAccessFile> splitFile(String srcFile) throws IOException {
        Integer perFileHoldNum = TOTAL_NUMBER/SPLIT_NUMBER;
        RandomAccessFile file = new RandomAccessFile(srcFile,"r");
        List<RandomAccessFile> splitFileList = new ArrayList<>(SPLIT_NUMBER);
        List<Integer> dataList = new ArrayList<>(perFileHoldNum);
        String line;
        int fileIndex = 0;
        while((line = file.readLine()) != null){
            dataList.add(Integer.valueOf(line));
            //达到上限后写入一个文件
            if (dataList.size() == perFileHoldNum){
                dataList.sort(null);
                RandomAccessFile splitFile = new RandomAccessFile("./test/splitFile_" + fileIndex + ".txt","rw");
                for (Integer data : dataList){
                    splitFile.writeBytes(data + System.lineSeparator());
                }
                splitFileList.add(splitFile);
                dataList.clear();
                fileIndex++;
            }
        }
        return splitFileList;
    }
    //对文件进行排序,最终结果存储在resultFile中
    public static String mergeFile(List<RandomAccessFile> fileList) throws IOException {
        String resultFileName = "./test/resultFile.txt";
        if (Objects.isNull(fileList) || Objects.equals(fileList.size(), 0)) {
            return null;
        }
        if (Objects.equals(fileList.size(), 1)) {
            RandomAccessFile resultFile = reNameFile(fileList.get(0), resultFileName);
            resultFile.close();
            return resultFileName;
        }

        List<FileHolder> fileHolderList = new ArrayList<>();
        for (RandomAccessFile file: fileList){
            fileHolderList.add(new FileHolder(file));
        }

        RandomAccessFile result = new RandomAccessFile(resultFileName,"rw");
        String writeData;
        do {
            Collections.sort(fileHolderList, (o1, o2) ->{
                if (!isNotEmpty(o1.data)) {
                    return 1;
                }
                if (!isNotEmpty(o2.data)) {
                    return -1;
                }
                Integer firstNumber = Integer.valueOf(o1.data);
                Integer secondNumber = Integer.valueOf(o2.data);
                return firstNumber.compareTo(secondNumber);
            } );
            FileHolder fileHolder = fileHolderList.get(0);
            writeData = fileHolder.data;
            if (isNotEmpty(writeData)){
                fileHolder.data = fileHolder.file.readLine();
                result.writeBytes(writeData + System.lineSeparator());
            }
        } while (isNotEmpty(writeData));

        for (RandomAccessFile file: fileList){
            file.close();
        }

        result.close();
        return resultFileName;
    }

    public static Boolean isNotEmpty(String str) {
        return str != null && str.length() > 0;
    }

    //测试(在项目根目录下新建test文件夹)
    public static void main(String[] args) throws IOException {
        String fileName = "./test/source.txt";
        wirteFile(fileName);
        mergeFile(splitFile(fileName));
    }
}
